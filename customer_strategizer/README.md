# 🎯 Customer Churn & LTV Strategizer

An end-to-end data science project featuring a machine learning pipeline that predicts customer churn risk and evaluates behavior patterns, complete with an interactive web dashboard.

## 🚀 Project Overview
This project demonstrates how machine learning can be used to solve real-world business retention problems. Using a Random Forest classifier trained on a telecom dataset, this application calculates the percentage probability that a given customer will stop doing business with a company.

## 🛠️ Tech Stack
* **Language:** Python 3.12
* **Libraries:** Pandas, NumPy, Scikit-Learn, Joblib
* **Interface:** Streamlit (Web App framework)
* **Environment:** VS Code & Git

## 📊 Features
* **Machine Learning Brain:** Powered by a Random Forest Classifier trained on consumer behavior.
* **Probability Scoring:** Instead of a simple Yes/No, it outputs the exact mathematical risk of churn.
* **Interactive UI:** Sliders and inputs allowing users to simulate customer behavior and see risk change in real time.

## ⚙️ How to Run Locally
1. Clone the repository:
   `git clone https://github.com/RavinduBatawala/customer-churn-strategizer.git`
2. Navigate to the folder:
   `cd customer-churn-strategizer`
3. Activate the virtual environment and install requirements:
   `.\venv\Scripts\activate`
   `pip install -r requirements.txt`
4. Run the app:
   `python -m streamlit run app.py`
